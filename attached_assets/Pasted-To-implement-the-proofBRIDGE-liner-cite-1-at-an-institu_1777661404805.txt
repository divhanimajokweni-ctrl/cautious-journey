## 1️⃣ The Coq Extraction & Soundness Proof
We move beyond pseudo-proofs into a structural formalization. The goal is to ensure the **Safety Kernel**[cite: 1] is a total function where the `HALTED` state is absorbing.

```coq
(* File: SafetyKernel.v *)
Require Import QArith.

Inductive State := OPEN | HALTED.
Inductive Actor := AUTH | UNAUTH.

Record KernelInput := {
  posterior : Q;
  threshold : Q;
  actor : Actor
}.

Definition step (s : State) (i : KernelInput) : State :=
  match s with
  | OPEN => if Qle_bool i.(threshold) i.(posterior) then HALTED else OPEN
  | HALTED => match i.(actor) with
              | AUTH => OPEN
              | UNAUTH => HALTED
              - end
  end.

Theorem unauthorized_halt_is_absorbing : 
  forall (i : KernelInput), i.(actor) = UNAUTH -> step HALTED i = HALTED.
Proof. intros i H; simpl; rewrite H; reflexivity. Qed.
```
