## 1. Core Implementation Components

### 1.1 Stratified Probabilistic Scorer (prover/scorer.js)
- Function: Computes Beta-Binomial posterior mean with scenario classification
- Inputs: Gateway results array, expected hash
- Outputs: Trigger score, scenario (A/B/C), threshold used, trip decision
- Prior: Configurable Beta(α, β) from config/scoring.json
- Classes:
  - A: ≤1 mismatch, ≥2 responders → τ_A = 0.60 (weak evidence)
  - B: ≥2 mismatches, ≥2 responders → τ_B = 0.355 (strong evidence)
  - C: 0 responders → no trip (handled by unreachable logic)

### 1.2 Configuration System (config/scoring.json)
{
  "priorAlpha": 1,
  "priorBeta": 10,
  "thresholdA": 0.60,
  "thresholdB": 0.355,
  "minMismatchesB": 2,
  "minGatewaysB": 2
}

### 1.3 Submitter Integration (prover/submitter.js)
- Modified planActions() to use scoreAsset() for trip decisions
- Trip reasons include scenario and threshold: "Stratified trigger: scenario B, score 0.400 ≥ 0.355"

### 1.4 Monte Carlo Simulation Suite
- ROC Calibration (test/simulate-classification.js): Computes optimal τ* at γ=10
- Adversarial Validation (test/adversarial-sim.js): TPR/FPR under k=0-4 compromised gateways
- Threshold Recalibration (test/recalibrate-thresholds.js): Recomputes τ_A, τ_B with empirical priors

### 1.5 Empirical Calibration Pipeline
- Harness (test/calibrate-prior.js): Estimates false mismatch rate μ across cycles
- Aggregator (test/aggregate-calibration.js): Smoothed prior estimation with (1 + mismatches)/(2 + observations)
- Automation (test/automate-calibration.sh): End-to-end multi-CID workflow

## 2. Mathematical Foundations

### 2.1 Probabilistic Model
- Prior: Beta(α₀=1, β₀=10), mean μ₀ = 0.0909 (conservative validity bias)
- Likelihood: Binomial(mismatches | responders, θ), θ unknown tampering probability
- Posterior: Beta(α₀ + mismatches, β₀ + responders - mismatches)
- Trigger Score: Posterior mean = (α₀ + m)/(α₀ + β₀ + r)

### 2.2 Stratified Thresholds
- Class A (Weak Evidence): τ_A = 0.60, requires stronger evidence for trip
- Class B (Strong Evidence): τ_B = 0.355, trips on lower posterior probability
- Cost Ratio: γ=10 (false negative 10× costlier than false positive)
