# Crisis Response Playbook - ProofBridge Liner

## Incident Response Steps
1. Detection: Kernel flags high-risk event.
2. Containment: Block transaction.
3. Notification: Alert FSCA, CISO.
4. Investigation: Package forensic evidence.
5. Recovery: Resume with fallback mode.
6. Reporting: Submit SAR and incident logs.

## Escalation Matrix
- Low risk: Log and proceed.
- High risk: Halt and notify regulators.
- Critical: Full system lockdown.
