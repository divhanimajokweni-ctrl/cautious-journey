# Disaster Recovery - ProofBridge Liner

## Failure Scenarios
- TEE compromise: Pivot to gamma=50 fallback.
- API outages: Hybrid Deeds API with caching.
- Network issues: Offline mode with queued reports.

## Recovery Procedures
1. Assess TEE integrity.
2. Switch to fallback gamma.
3. Resume operations with manual oversight.
4. Generate post-incident reports.
